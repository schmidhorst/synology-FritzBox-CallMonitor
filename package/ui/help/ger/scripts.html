DSM Shell-Skripte

Jedesmal wenn die Fritzbox einen Anrufänderung meldet, kann das angegebene Skript ausgeführt werden. Anrufänderungen sind RING (es klingelt), CALL (nach dem Wählen), CONNECT (angerufener hat abgenommen) oder DISCONNECT (Ende, auch z.B. Abbruch)
Das BASH-Skript wird als user root ausgeführt. Zu beachten ist, dass nicht der Windows-übliche Zeilenumbruch <CR><LF> und auch nicht der Apple-übliche <CR> verwendet werden darf, sondern nur der Linux-Zeilenumbruch <LF> funktioniert.
Dem Skript werden als Parameter <what> <externe Nummer> <externer Name> <TelefonbuchName> <Line-Nummer> <NebenstellenNr oder ~Name> übergeben. Das <what> ist CALL, RING, CONNECT oder DISCONNECT
