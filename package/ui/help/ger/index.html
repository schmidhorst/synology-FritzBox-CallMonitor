Callmonitor

Das "Callmonitor"-Paket wurde entwickelt für ein Heimnetz mit einer AVM-Fritzbox und einer HomeMatic-CCU. Der Callmonitor reagiert auf die über die Telefonfunktion der FritzBox geführten Telefonate.

Das Paket schreibt Angaben zum Anfuf in Systemvariablen der CCU, wenn ein Telefonanruf reinkommt oder rausgeht. Am Gespächsende wird der Systemvariablen-String dann wieder gelöscht. Mit einem Programm oder Skript in der CCU kann auf diese Änderungen reagiert werden.

Über eingebundene Adressbücker können die Rufnummer des Anrufers, die Nummer VoIP-Leitung und die Nummer des internen Telefoniegeräts zu Namen aufgelöst werden.

Die Anrufe werden auch protokolliert und diese Anruflisten (Eingehend, Ausgehend, Verpasst, Alle) können auf Webseiten der DSM eingesehen werden.

Für Konfigurationsänderungen des Callmonitors bitte die Installation erneut ausführen! Dabei werden die bisherigen Konfigurationswerte vorgegeben.
