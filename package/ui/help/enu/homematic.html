HomeMatic-CCU

If you want to use this feature, then for each used external line number of the FritzBox you need to specify the name of as CCU system variable (Type "String"). At the Callmonitors installation you need to specify access data for the CCU (IP address, Username, Password).

Security Hint: The CCU-Password is stored in clear, not encrypted

At begin of a call somthing like the following will be writen to that variable:
IN;CallerNummer;CallerName;BookName;OwnNummer
OUT;CalledNummer;CalledName;BookName;OwnNummer;UsedOwnExtension
At the ende of the call the system variable will be set to an empty string.
