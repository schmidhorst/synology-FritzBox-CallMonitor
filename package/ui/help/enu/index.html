Callmonitor

The "Callmonitor" was developed for an home net with an AVM-Fritzbox and a HomeMatic-CCU. The Callmonitor is reaction to phone calls done via the FritzBox.

System variables of the CCU will be set at the start of an phone call. At the end of the call the variable content will be deleted. With an CCU programm or script you can react on that.

Via configured address books the caller numbers can be converted to names.

The calls are recorded and those records are available via web paged.

For configuration changes please run the Callmonitor installation again!
