DSM Shell-Skripte

Each time, the Fritz!Box notices for a phone call change (RING, CALL, CONNECT, DISCONNECT), the script will be called.
The script is executed as user root. Parameters are <what> <externe Nummer> <externer Name> <TelefonbuchName> <Line-Nummer> <NebenstellenNr oder ~Name>. The <what> is CALL, RING, CONNECT or DISCONNECT.
